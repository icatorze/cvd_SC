Written in Python
## Depends: 
 - BeautifulSoup
 - ast
 - slimit
 - json
 - pandas
 - matplotlib
 - numpy


## Features

- get_data.py - Collect data
- plot_data.py - Plot 2D graphics 
- death_data - Plot 3D graphics of deaths based on Date - Age - number of deaths.
